# CC3085-Proyecto3 - Clasificador SPAM/HAM con Redes Bayesianas

## Requisitos previos

- **Python 3.8+**
- **pip** 

## Instalación de dependencias

Ejecuta el siguiente comando para instalar todas las librerías necesarias:

```bash
pip install pandas numpy matplotlib wordcloud nltk scikit-learn scipy
```

### Dependencias principales:
- `pandas` - Manipulación de datos
- `numpy` - Operaciones numéricas
- `matplotlib` - Visualizaciones
- `wordcloud` - Nubes de palabras
- `nltk` - Porter Stemmer para procesamiento de texto
- `scipy` - Cálculo de densidades (KDE)

## Cómo correr el programa

### Terminal 

```bash
python proyecto3.py
```

En **Windows PowerShell**, si tienes problemas, usa:

```bash
& "path/to/python.exe" proyecto3.py
```
